Project Overview

This project is a face recognition-based attendance system. It allows you to take attendance by detecting faces through a webcam. It uses a MySQL database to store attendance records, and it leverages the power of OpenCV, MediaPipe, and facial recognition algorithms.

Modules Used

  • OpenCV: Used for computer vision tasks, including face detection and image processing.
  • MediaPipe: Provides an easy-to-use face detection model that helps in detecting faces from the camera feed.
  • MySQL Connector: Connects the Python code to the MySQL database to save attendance data.
  • pickle: Used for creating and loading the trained face recognition model.
  • sklearn: A library for preprocessing the labels (student names) for the face recognition model.MACHINE LEARNING GOESSSS WOOOOOOOOOOOOOOO

  1. Install all necessary modules by running the following command:
    2. pip install opencv-python mediapipe mysql-connector-python scikit-learn
  2. Make sure you have a MySQL server running locally. If not, you can download and install MySQL from the official website.
  3. Import the database schema from the attensql.sql file. This will set up the required tables in your database.
  4. Modify the MySQL connection parameters in the code (username, password, and database name) to match your setup.
  5. Run the Python script that starts the face recognition system. It will prompt you to enter the student's name for training and allow you to take attendance.

What This Project Does

This project performs the following tasks:

  • Face Detection: Detects faces using MediaPipe's face detection module.
  • Face Recognition: Recognizes trained faces with OpenCV's LBPH (Local Binary Patterns Histograms) face recognizer.
  • Attendance Logging: Logs the name of the recognized person into a MySQL database.
  • Admin Authentication: Allows an admin to authenticate with a username and password before using the system.
  • Real-Time Monitoring: Displays the webcam feed with detected faces and logs attendance when a face is recognized.
